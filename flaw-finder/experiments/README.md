# Experiments

Use this folder as the lab notebook. Every experiment should be small enough to
run in a day and honest enough to kill a weak assumption.

## Template

```markdown
# YYYY-MM-DD - Experiment name

## Hypothesis

What must be true?

## Method

What will we build, show, or measure?

## Success signal

What result means we keep going?

## Result

What happened?

## Decision

Keep, change, or stop?
```

## First experiments

1. **Manual report test**: Run 5 founder pitches through the local pipeline,
   manually improve the report, and track which feedback people find useful.
2. **Persona usefulness test**: Compare investor/lawyer/user personas against
   alternate roles like buyer, competitor, or security reviewer.
3. **Input format test**: Try pasted text, uploaded deck text, and website copy
   to learn which format produces the strongest first impression.
